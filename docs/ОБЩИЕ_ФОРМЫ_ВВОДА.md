# Общие формы ввода Wallet

## Выбор счёта

`ОбщаяФорма.ФормаВыбораСчета`

```bsl
ОбщийМодульКлиент.ПоказатьВыборСчета(ОписаниеОповещения, ТекущийСчет, Заголовок, ИсключитьСчет);
```

Возвращает `СправочникСсылка.Счета` или `Неопределено`.

## Текстовая клавиатура

`ОбщаяФорма.ФормаТекстовойКлавиатуры`

```bsl
ОбщийМодульКлиент.ПоказатьВводТекста(ОписаниеОповещения, Текст, Заголовок, МаксДлина);
```

Возвращает строку или `Неопределено`. Раскладки: RU / EN / 123, Shift, пробел, ⌫, Очистить, Готово.

## Где подключено

- Перевод: выбор счетов и комментарий
- Долг: комментарий
- (цифровая клавиатура суммы — см. `ЦИФРОВАЯ_КЛАВИАТУРА.md`)
