# HTML-дашборд начального экрана

Дата: 2026-07-18  
Подход: гибрид — только главная страница в HTML, остальное через формы 1С.

## Что сделано

- Форма `НачальныйЭкран` показывает главную в `ПолеHTML` (`ТекстHTML`).
- HTML-каркас встроен в `ТекстМакетаГлавнойСтраницы()` модуля формы (без `ПолучитьМакет` — так надёжнее при обновлении ИБ).
- Копии вёрстки для правки: `docs/ui/dashboard.html`, `Templates/МакетГлавнойСтраницы/Template.txt` (после правки перенести строки в функцию модуля).
- Клики по ссылкам `app:…` → существующие формы/отчёты.
- Страница «Отчёты» на управляемой форме сохранена.
- Рабочая область начальной страницы: `Configuration/HomePageWorkArea.hpwa` → форма обработки.

## Команды HTML

| Ссылка | Действие |
|--------|----------|
| `app:rashod` | `ОбщаяФорма.ОбщаяФормаДокумента` (расход) |
| `app:dohod` | `ОбщаяФорма.ОбщаяФормаДокумента` (доход) |
| `app:reports` | страница отчётов формы |
| `app:accounts` | список `Справочник.Счета` |
| `app:categories` | список `Справочник.КатегорииРасходов` |
| `app:period:Расходы\|Доходы:Сегодня\|Неделя\|Месяц` | отчёт за период |
| `app:account:<УИД>` | отчёт по взаиморасчётам по счёту |

## Как править внешний вид

1. Править `docs/ui/dashboard.html` (или `Templates/.../Template.txt`).
2. Перенести изменения в `ТекстМакетаГлавнойСтраницы()` модуля формы.
3. Плейсхолдеры: `#Подзаголовок#`, `#СтрокиСводки#`, `#ИтогСводки#`, `#СтрокиСчетов#`.
4. Данные подставляет `СобратьHTMLГлавнойСтраницы()`.

## Проверка на устройстве

1. Обновить конфигурацию МП / пересобрать приложение.
2. Главный экран ≈ макету (`docs/ui/mobile-ui-mockup-uchet.png`).
3. Расход/Доход → быстрый ввод; после записи сводка обновляется.
4. Счета / Категории / Отчёты / строки сводки и счетов — переходы работают.
5. Если клик не ловится: смотреть `ДанныеСобытия.Href` (должен содержать `app:`).
