# Этап 11. Переводы между счетами

Цель: документ перемещения денег между счетами без влияния на доходы/расходы, с корректным влиянием на остатки.

Источник: [Этап 11 — Переводы между счетами](https://app.notion.com/p/b72fb95f9e044a9bb544495a1311f0a1).

## Реализация

### Документ `Переводы`

Реквизиты:
- `СчетИсточник` — счёт списания;
- `СчетПолучатель` — счёт зачисления;
- `Сумма`, `Валюта`;
- `Комментарий` (причина) — необязательный.

Проведение — только РН `Баланс`:
- **Расход** по счёту-источнику;
- **Приход** по счёту-получателю.

Движений по `Доходы` / `Расходы` нет → переводы не попадают в отчёты доходов и расходов.

Валидации: оба счёта обязательны и различны, сумма > 0, курс валюты при необходимости.

### UI

- Кнопка **Перевод** на главной (`app:perevod`) открывает форму документа.
- Меню → **Переводы** — список документов.
- После закрытия формы главная обновляет остатки.

## Критерии готовности

- [x] Документ создан и доступен с главной / из меню.
- [x] Движения по регистру баланса.
- [x] Переводы влияют на остатки счетов.
- [x] Переводы не пишутся в регистры доходов/расходов.
- [x] Главная обновляется после перевода.
