# Цифровая клавиатура Wallet

Общая форма `ОбщаяФорма.ФормаЦифровойКлавиатуры` по разделу «Цифровая клавиатура» дизайн-системы Wallet.

## API

```bsl
ОбщийМодульКлиент.ПоказатьВводСуммы(ОписаниеОповещения, Сумма, Заголовок, Режим);
```

- `Режим`: `income` (зелёный), `expense` (красный), `transfer` (синий)
- В оповещение приходит `Число` или `Неопределено` (отмена)

## Где подключено

- Перевод между счетами (HTML)
- Быстрый ввод прихода/расхода (`ОбщаяФормаДокумента`)
- Документы `Доходы`, `Расход`, `ПередачаСредствВДолг`
