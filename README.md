# Учет расходов и доходов

Мобильное приложение на платформе **1С:Предприятие 8.3** для личного учёта финансов.

| Параметр | Значение |
|---|---|
| Имя конфигурации | `УчетРасходовИДоходов` |
| Версия | 1.0.2 |
| Платформа / совместимость | 8.3.22 |
| Назначение | Мобильное устройство (`MobileDevice`) |
| Режим | Управляемое приложение |
| Автор | Досщанов Серикбай, г. Алматы, 2022 |

## Назначение

Приложение позволяет вести персональный учёт доходов и расходов:

- фиксировать операции по счетам (наличные, банковская карта, банковский счёт);
- классифицировать операции по категориям доходов и расходов;
- видеть остатки по счетам и сводку за сегодня / неделю / месяц на начальном экране;
- анализировать данные через отчёты по доходам, расходам и взаиморасчётам;
- при первом запуске задавать валюту учёта.

## Основные объекты

### Справочники
- **Счета** — кошелёк / карта / банковский счёт (предопределённые элементы)
- **КатегорииДоходов**, **КатегорииРасходов** — иерархические категории
- **Валюта** — валюты операций

### Документы
- **Доходы** — приход денежных средств → регистры `Доходы`, `Баланс`
- **Расход** — расход денежных средств → регистры `Расходы`, `Баланс`

### Регистры
- Накопления: `Баланс`, `Доходы`, `Расходы`

### Отчёты
- `ОтчетПоДоходам`, `ОтчетПоРасходам`, `ОтчетПоВзаиморасчетам`

### Обработки и обмен
- **НачальныйЭкран** — главный экран приложения (сводка и счета)
- **ЗапаснойОбменДанными** — резервный обмен
- План обмена **Мобильные** — обмен с мобильными устройствами

### Общие модули
- `ОбщийМодульКлиент` — клиентский UI и вызовы операций
- `ОбщийМодульСервер_СистемныеНастройки` — первый запуск, валюта, предопределённые счета
- `ОбщийМодульСервер_УправлениеФормами` — программное создание элементов форм
- `ОбщийМодульСервер_ОперацииУчета` — создание и проведение операций

Проект целиком на языке **BSL (1С)**, без других языков. При доработках ориентироваться на мобильный UX: компактные формы, цифровая клавиатура (`ФормаЦифры`), календарь (`ФормаКалендаря`), адаптация шрифтов под экран.
